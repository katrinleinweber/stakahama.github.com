Setting up

Directory structure

Emacs Wiki says that the ~/.emacs.d/ directory is for add-on packages, etc., but this has only been a recent development. Many old-time emacs users roll their own package directory (e.g., Bill Clementson's setup for clojure), like ~/elisp. I've moved everything to ~/.emacs.d/ to include all customization files: user-configured files, lisp code, and standard packages.

These are all added to my path with (add-to-list 'load-path "~/.emacs.d") and so on.

Package types

Can be

  1. distributed a single file or directory of emacs-lisp files
  2. distributed as a collection of files (emacs lisp and other) with a config and makefile with typical args:
    • --with-emacs
    • --with-prefix
  3. ELPA (now distributed with Emacs 24)

The type also affects portability across machines.

  • The first type - use M-x byte-compile-file or M-x byte-recompile-directory. Use C-u M-x byte-recompile-directory or C-u 0 M-x byte-recompile-directory.
  • Second type: compile as desired
  • Third type: …(no experience)

Some of these are available through the Debian repository.

Customizations

Key bindings

As a user, you can redefine any key, but it is usually best to stick
to key sequences that consist of `C-c' followed by a letter (upper or
lower case). These keys are "reserved for users," so they won't
conflict with any properly designed Emacs extension. The function keys
F5 through F9 are also reserved for users. If you redefine some other
key, your definition may be overridden by certain extensions or major
modes which redefine the same key.

There are many ways in which people have customized their keybindings

Changing custom variable values

Note that t and values zero or greater are used to indicate non-nil values (0 is equivalent to t for on/off t/f purposes); negative values are equivalent to nil. 

(cua-mode nil)
(setq delete-active-region nil)
(delete-selection-mode nil)
(setq ring-bell-function 'ignore)
(blink-cursor-mode nil)
(normal-erase-is-backspace-mode t)
(show-paren-mode t)
(setq inhibit-splash-screen t)
(setq initial-scratch-message nil)
(setq line-number-mode t)
(setq column-number-mode t)
(mouse-wheel-mode t)                    ; activate mouse scrolling
(global-font-lock-mode t)               ; syntax highlighting
(tool-bar-mode -1)
(fset 'yes-or-no-p 'y-or-n-p)
