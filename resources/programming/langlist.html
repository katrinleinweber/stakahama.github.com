R

Positives:

  • as DSL, easy to install and maintain (and install new libraries)
  • idiom of statisticians: 1-indexing, data frame object and operations
  • statistical libraries
  • dated graphics setting defaults
  • high-level plotting libraries (lattice, ggplot2)
  • interface with SQL and other tools

Downsides:

  • bad memory management ("smart" copy-by-value)

Fortran

77, 90 […] 2008

Pros

Widespread use:

  • chemical transport models
  • chemical mechanisms
  • DL_POLY, AMBER
  • NAMD is written in C++ but is not meant to be modified by the casual user anyway

Modern fortran has incorporated many of the convenient modularity of C++ using objects and namespaces.

Cons

  • Strings: Still bad at string manipulation though as pointed out by erict.
  • Modules: Importing module with USE statement dumps everything in global namespace, but possible use of ONLY statement. See this discussion on a compromissory convention.
  • Array indexing: The parentheses for array indexing is confusing as well as it can be mistaken for a function call. One interpretation is that the variable indexing is a look-up function call but in a practical sense you have to go back and check to see if it is a declared variable in that subroutine/function or in one of the modules which is loaded.
