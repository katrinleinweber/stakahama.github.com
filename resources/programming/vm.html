1 Install Ubuntu Linux as guest

Taken from reference this forum:

Run /media/VBOXADDITIONS_*/VBoxLinuxAdditions.run to install (if you update packages, you may have to rerun it). 

sharename=VirtualBoxDocuments \
mountpoint=~/$sharename \
sudo mkdir $mountpoint \
sudo chmod 777 $mountpoint \ ##important, otherwise error: "not a block device"
sudo mount -t vboxsf -o uid=1000,gid=1000 $sharename $mountpoint

For the above command if you get error as 

mount: unknown filesystem type 'vboxsf'

Then just change the vboxsf to vboxfs

If you want to have it mount automatically upon each boot, put the mount command in /etc/rc.local Debian distros (e.g. Ubuntu 10.04 and later): 

sudo mount -t vboxsf -o uid=1000,gid=1000 $sharename $mountpoint
