Jupyter

Jupyter requires python 3+ and python 2.7. I already had conda (python 2.7) installed.

Following instructions from the Revolution Analytics blog: 

$ conda install -c ipython-notebook r-irkernel ipython notebook

Note that I skipped installation of conda r as I already had this installed separately. 

$ which R
/usr/local/bin/R

Next, I follow the instructions from the IRkernel page: 

> install.packages(c('pbdZMQ', 'repr', 'devtools'))  # repr is already on CRAN
> devtools::install_github('IRkernel/IRdisplay')
> devtools::install_github('IRkernel/IRkernel')
> IRkernel::installspec()  # to register the kernel in the current R installation

To begin a session with R: 

$ jupyter notebook --kernel=ir
