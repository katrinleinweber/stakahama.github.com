Collection of code, software, and scripts

My repositories are on GitHub and Bitbucket.

  • pmf (R, public, GitHub): A set of scripts for generating solutions using Penti Paatero's PMF2 executable. Has been used with FTIR, STXM, AMS, and ACSM data sets. Also includes scripts for pixel-level cluster analysis of STXM stacks.
  • pscf (R, Python, and LaTeX; public, GitHub): My implementation of Positive Matrix Factorization (originally by Phil Hopke).
  • semsdat2itx (Python, public, GitHub): A script to convert BMI SEMS .dat files to Igor Pro .itx files.
  • dot-emacs (Emacs-Lisp, public, GitHub): My emacs configuration files.
  • bibfile (LaTeX-bib, public, GitHub): My .bib file for LaTeX documents. Generated and managed through JabRef.
  • rlib (R, public, GitHub): A set of R functions.
  • mlib (MATLAB, public, GitHub): A set of MATLAB functions.
  • pylib (Python, public, GitHub): A set of Python functions.
  • igorlib (Igor Pro, public, GitHub): Igor Pro functions and subroutines.
  • r-tutorial (LaTeX and R, public, GitHub): An R tutorial I wrote back in 2008.
  • curriculum-vitae (LaTeX, public, GitHub): My compiled curriculum vitae in pdf. LaTeX source files are hosted in a private repository (curriculum-vitae).
  • ftir (R, private, Bitbucket): A set of scripts for analysis of infrared absorption spectrum of atmospheric particles collected on Teflon filters. Team: aerosolftirusers.
  • stxm (MATLAB, private, Bitbucket): A set of scripts for image analysis and statistical learning for STXM-NEXAFS stacks. Team: stxmusers.
  • stxmclass (R, private, Bitbucket): A set of scripts for classification of NEXAFS spectra and preparation of matrices for PMF analysis (pmf).
  • myshell (Bash and Python, private, Bitbucket): A set of scripts to generate my custom .bashrc, .bash_aliases, .bash_profile for each OS.
  • curriculum-vitae (LaTeX, private, Bitbucket): My curriculum vitae in LaTeX. The compiled pdf is pushed to public GitHub repository of the same name, curriculum-vitae.
  • orgpages (Emacs-Lisp, private, Bitbucket): My Emacs org-files used to generate these webpages (http://stakahama.github.com, stakahama.github.com).
