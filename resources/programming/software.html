Package managers on Mac

Three main package managers for third-party software for Mac are Homebrew, Macports, and Fink. My choice in using Macports was mostly a determined by time of adoption since the development Fink was uncertain and Homebrew did not exist. Homebrew is trending in popularity, but the following post summarizes my decision to stay with Macports most rationally: http://terrychay.com/article/macports-vs-homebrew.shtml.

Homebrew apparently uses modern tools which make writing package installers (using Ruby) more easy than Macports–but for a person heavy on the user end, Macports is more than sufficient. Though it's not ideal that Macports installs duplicate libraries which come with OS X, hard disk space is (relatively) cheap and the number of packages offered by Macports makes it a practical choice.
